### dont forget to create a .env file and in there write this:

`PORT = 3001`

`NODE_ENV = development`

`CLIENT_URL = *`

### after that you can run the server with this command: `npm start`

### also before running npm start dont forget to run `npm i express dotenv socket.io`

### also before running dont forget to close any other server running on port 3001

and dont forget to change the `CLIENT_URL` to your own server's url
